# 関節トラッキングシステム

このプロジェクトは、MOVファイルなどのビデオから人体の関節を検出し、その動きをトレースして視覚化するコマンドラインツールです。MediaPipeとOpenCVを使用して高精度な姿勢推定を実現します。

## 機能

- MediaPipeを使用した高精度な姿勢検出
- 主要な関節（手首、肘、膝、足首など）の軌跡の可視化
- 処理結果をMP4ビデオファイルとして保存
- リアルタイムプレビュー機能（オプション）
